We created a two-sided card using the box style that effectivley distinguishes between information and branding. While the back side is neat and organized for easy contact reference, the front side highlights identity and style with a striking header and image backdrop. Strong color contrast a distinct font structure, and simple CSS layout properties give it that polished appearence. And most importantly in our opinion, fiery orange hues combined with the background image using background-blend-mode: color-burn produce a flaming dynamic appearence that emphasizes the brand's athletic and lively spirit. It also goes with the slogan, "BEYOND THE BURN."

We created a one-sided card using the flex style that uses horizontal flex sections for balanced spacing at the bottom and vertical stacking of elements for a neat top-to-bottom flow. Flex properties manage distribution and alignment, resulting and aesthetically pleasing layout. Similar to a grid but more flexible, the style makes use of column-based parent container with nested horizontal flex rows, enabling vertical stacking for structure and horizontal grouping for exact spacing. In addition to adding personality and that royal look, the bold color scheme of gold, purple, and green on a black background and the similar font produce a polished and unified appearence.
